okapi::IterativeVelocityControllerArgs
======================================

Data class for the arguments to ``IterativeVelocityController``.

okapi::IterativeVelocityController
==================================

An ``IterativeVelocityController`` is a type of `IterativeController <abstract-iterative-controller.html>`_ that
operates in the velocity domain.
